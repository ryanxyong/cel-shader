Task 4: Cel Shading - Ambient/Scene colors

We refer to this video by Jasper for some of the color values used in this section.

We have modified the toon_shade_color function to store a value if the current vertex is lit up by light by utilizing the existing conditions of whether or not a luminosity value is a shadow or not (stored in boolean isHighlight). For each condition, we call calc_vertex_color, which takes in isHighlight and uses an integer Mode value stored as a constant to determine time of day, to return the appropriate color for the shadows and highlights of an object given the time of day. These colors are based on those that are used in Zelda: The Wind Waker.

We still manage to employ the section of smooth gradient in the very center of the luminosity change in toon_shade_color when the vertex is not a shadow nor a fully highlighted vertex by scaling down the highlight color and adding it to the shadow color.
